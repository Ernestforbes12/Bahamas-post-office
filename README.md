# 🇧🇸 Bahamas Post Office Redesign

A modern, fully responsive redesign of the official Bahamas Postal Service website, built with React, Tailwind CSS, Three.js, and Framer Motion.

## 🔗 Live Demo
[View Live Site](https://bahamas-post-office.netlify.app)

## 📸 Before & After
| Before | After |
|--------|-------|
| Original bahamaspostal.gov.bs | Modern React redesign |

## ✨ Features
- 🌍 3D rotating Earth globe built with Three.js and React Three Fiber
- 📦 Package tracking bar with cyan glow animation
- 🎴 Service cards with 3D flip effect on hover
- 📊 Animated stats counter triggered on scroll
- 🧮 Interactive rate calculator (Local vs International)
- 📍 Branch locator across the Family Islands
- 📬 Contact form
- ☰ Fully responsive with mobile hamburger menu
- 🎬 Framer Motion animations throughout

## 🛠️ Built With
- React
- Tailwind CSS
- Framer Motion
- Three.js
- React Three Fiber
- React Three Drei
